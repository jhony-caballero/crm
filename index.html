Conectar Firebase

Sincroniza datos entre celular y computadora en tiempo real.

1
Ir a Firebase Consoleconsole.firebase.google.com → inicia sesión con Google.
2
Crear proyecto"Agregar proyecto" → nombre → desactiva Analytics → Crear.
3
Activar FirestoreMenú → Firestore Database → Crear → Modo prueba → us-central1 → Listo.
4
Activar AuthenticationMenú → Authentication → Comenzar → Correo/Contraseña → Habilitar → Agregar usuario.
5
Copiar credenciales⚙️ Configuración → General → Tus apps → </> → Config → copiar valores.
